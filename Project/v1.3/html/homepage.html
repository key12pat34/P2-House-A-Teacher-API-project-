Project Summary

The purpose of this project was to analyze how weather changes as you get closer or move away from the equator. Using python and APIs, over 500 cities from across the world were used in the dataset.

After assembling the dataset, various plots were created to show different visualizations and trends, and be able to explain how cities are affected based on their location to the equator.
